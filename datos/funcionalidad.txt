Funcionalidades Bootstrap utilizadas en el Proyecto:

1. **Sistema de Rejilla:**
   - Se utilizó el sistema de rejilla de Bootstrap para diseñar el diseño responsivo de las páginas.
   - Se emplearon las clases `container` y `container-fluid` para envolver y ajustar el contenido.
   - Se utilizaron clases como `col-xxl`, `col-lg`, `col-md`, `col-sm`, etc., para definir el diseño en diferentes tamaños de pantalla.

2. **Barra de Navegación:**
   - La barra de navegación se implementó con el componente `navbar` de Bootstrap.
   - Se incluyeron elementos como `navbar-brand`, `navbar-toggler`, y `collapse` para lograr una barra de navegación responsiva.

3. **Tarjetas de Producto:**
   - Las tarjetas de productos se diseñaron utilizando el componente `card` de Bootstrap.
   - Se incluyeron clases como `card-img-top`, `card-body`, `card-title`, `card-text`, y `btn` para estructurar y estilizar las tarjetas.

4. **Carousel:**
   - Se implementó un carrusel de imágenes utilizando el componente `carousel` de Bootstrap.
   - Se usaron clases como `carousel-inner`, `carousel-item`, y controles `carousel-control-prev` y `carousel-control-next`.

5. **Formulario:**
   - El formulario de contacto se diseñó utilizando las clases de formulario proporcionadas por Bootstrap.
   - Se utilizaron clases como `form-label`, `form-control`, y `btn` para estilizar los elementos del formulario.

6. **Botones y Estilos:**
   - Se emplearon clases de botón de Bootstrap, como `btn` y `btn-outline-dark`, para estilizar los botones en diversas partes del sitio.

7. **Modal:**
   - Se implementó un modal utilizando el componente `modal` de Bootstrap.
   - Se incluyeron clases como `modal`, `modal-dialog`, y `modal-content` para estructurar y estilizar el modal.

8. **Utilización de Íconos:**
   - Se utilizaron íconos de Bootstrap, como los de la biblioteca de iconos de Bootstrap (`bi`), para mejorar la presentación visual de ciertos elementos.

Breakpoints utilizados en la grilla:

- `col-xxl`: Pantallas extra extra grandes (mayores a 1400px).
- `col-lg`: Pantallas grandes (entre 992px y 1400px).
- `col-md`: Pantallas medianas (entre 768px y 992px).
- `col-sm`: Pantallas pequeñas (entre 576px y 768px).
- `col-xs` o `col`: Pantallas extra pequeñas (menores a 576px).

Página de Productos - Consigna Elegida:

- Se eligió utilizar una disposición en tarjetas (cards) para mostrar los productos en la página de productos.
- Cada tarjeta contiene una imagen del producto, el nombre, una descripción breve y un botón de "Comprar".
- Además, se utilizó un carrusel de imágenes y un formulario de contacto para mejorar la experiencia del usuario.
